Getting Started

Unzip the archive and find out the folder Beautyqueen-Files. There are all the template files in this folder.

The file and folder structure is something like this:

  1. /css — folder with css files.
  2. /fonts — folder with icon fonts.
  3. /images — folder with image files.
  4. /js — folder with Javascript files.
  5. php — email.php
  6. php — reservation.php

Upload the template files to server with the help of one of the FTP-clients like FileZilla.

		<link rel="stylesheet" href="css/color.css">

Fonts used

By default, the template loads Open Sans, Rareway Web font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.css" file in the "css" folder:

	
		/*====== Link Google Fonts API ========*/
